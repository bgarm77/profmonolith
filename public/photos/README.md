# Фото проектов

Положите сюда фотографии ЖК — код **автоматически** их подхватит (сервер читает папку).
Формат: `.jpg` / `.jpeg` / `.png` / `.webp` / `.avif`. Сортировка — по имени файла,
поэтому называйте `01.jpg`, `02.jpg`, … Первый файл (`01.*`) — обложка карточки.

```
public/photos/
├── gyurjyan/      # Гюрджян Каскад (флагман) — 6 фото
├── ani/           # Ани Премиум — 5 фото
├── leningradyan/  # Ленинградян Резиденс — 5 фото
├── firdus/        # Firdus Prime — 4 фото
└── rubinyants/    # Рубинянц Комплекс — 5 фото
```

Пока папка проекта пуста — на карточке и в лайтбоксе показывается стильный плейсхолдер,
а кнопка «Смотреть фото» скрывается. Как только добавите файлы и пересоберёте (`npm run build`)
или перезапустите dev-сервер — фото появятся автоматически, правки кода не нужны.

Рекомендуемый размер: длинная сторона 1600–2000 px, оптимизация не обязательна —
`next/image` сам сделает AVIF/WebP и нужные размеры.
